## 🆘 **Quick Troubleshooting**

### **"My prices aren't updating"**
✅ **Check**: WebSocket indicator (should be green)
✅ **Try**: Refresh page (Ctrl+R)
✅ **Wait**: Updates come every 1 minute

### **"Strategy calculations look wrong"**
✅ **Verify**: Current stock price is accurate
✅ **Check**: Expiration date is correct
✅ **Confirm**: Strategy type matches your expectation

### **"Can't add a ticker"**
✅ **Check**: Stock symbol is valid (e.g., AAPL not Apple)
✅ **Try**: Different symbol if one fails
✅ **Wait**: Market might be closed

### **"Page won't load"**
✅ **Check**: Internet connection
✅ **Try**: Different browser
✅ **Clear**: Browser cache and cookies
